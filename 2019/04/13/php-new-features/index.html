PHP7 具有很多的用于编写高性能、高效代码的新特性，同时也移除了一些历史版本中过时的特性,下面是各版本中一些比较常用的新特性：

5.6.X - 7.0.X 新特性

标量类型声明有两种模式: 强制 (默认) 和 严格模式。 现在可以使用下列类型参数（无论用强制模式还是严格模式）： 字符串(string), 整数 (int), 浮点数 (float), 以及布尔值 (bool)。它们扩充了PHP5中引入的其他类型：类名，接口，数组和 回调类型。
